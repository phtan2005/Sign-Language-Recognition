🤟 Real-time Sign Language Recognition & Quiz Game (YOLOv11)Dự án nhận diện ngôn ngữ ký hiệu (Sign Language) theo thời gian thực sử dụng mô hình YOLOv11. Không chỉ dừng lại ở việc nhận diện các ký tự đơn lẻ, ứng dụng còn tích hợp hệ thống ghép từ (Word Builder) và chế độ trò chơi trắc nghiệm (Quiz Mode) giúp người dùng học và luyện tập ngôn ngữ ký hiệu một cách trực quan, sinh động. Hệ thống được tối ưu hóa bằng Temporal Smoother để chống nhiễu và hướng tới tốc độ suy luận mượt mà theo thời gian thực (mục tiêu 60 FPS).✨ Tính năng nổi bật⚡ Nhận diện thời gian thực (Real-time Inference): Sử dụng YOLOv11 kết hợp PyTorch (hỗ trợ CUDA/FP16) cho tốc độ xử lý khung hình cao.🛡️ Temporal Smoothing: Xử lý nhiễu (flickering) bằng hàng đợi (deque) và cơ chế voting, giúp kết quả nhận diện ổn định và chính xác hơn khi người dùng di chuyển tay.📝 Chế độ Word Builder:Tự động ghép các chữ cái thành từ vựng hoàn chỉnh khi giữ đúng ký hiệu trong một khoảng thời gian nhất định.Hỗ trợ quản lý phiên (session), xóa ký tự lỗi, và lưu trữ từ vựng vào file word_history.json.🎮 Chế độ Quiz (Mini-game):3 mức độ khó: EASY (15s/câu), NORMAL (10s/câu), HARD (6s/câu).Hệ thống tính điểm, chuỗi combo (streak), đếm ngược thời gian và đánh giá rank (S, A, B, C, D) vào cuối game.🖥️ Giao diện HUD trực quan: Hiển thị FPS, lịch sử từ vựng, thanh trạng thái (progress bar) và các hiệu ứng overlay được vẽ trực tiếp bằng OpenCV.🛠️ Yêu cầu hệ thốngĐảm bảo bạn đã cài đặt Python 3.8+ và các thư viện sau:Bashpip install ultralytics opencv-python torch torchvision
Lưu ý: Để đạt hiệu suất tốt nhất (60 FPS), khuyến nghị sử dụng GPU hỗ trợ CUDA.🚀 Hướng dẫn cài đặt và sử dụngClone repository:Bashgit clone https://github.com/phtan2005/sign-language-yolov11.git
cd sign-language-yolov11
Chuẩn bị model:Đặt file trọng số mô hình YOLOv11 đã train của bạn (ví dụ: best.pt) vào cùng thư mục với file code. (Bạn có thể chỉnh sửa đường dẫn MODEL_PATH trong code nếu cần).Chạy ứng dụng:Bashpython realtime_detect.py
